基本数据类型

数据类型

  • 数字(number)
  • 字符串(string)
  • 布尔(boolean)
  • undefined
  • null
  • Symbol
  • bigint
  • object

数字(number)

  • 整数、浮点数
  • Infinity, -Infinity
  • NaN: 非数值，任何有关于NaN的运算返回结果都是NaN, 可以通过isNaN(value), 来判断一个值是否是NaN
  • Math对象, 对数字的一些操作

字符串(string)

  • String()函数可以将任何值转换为字符串

布尔

  • 任何字符串可以转换为true/false
  • ‘’、0、null为false, 其余字符串，数字，对象为true

null和undefined

  • 俩者都表示空值
  • null：空对象的指针, undefined: 未经初始化的变量

symbol

  • 一般被用作一个私有的对象属性的键值
  • Symbol实例是唯一且不可改变的
  • Symbol()

bigint

  • 用来表示大于js不可表示的整数
  • BigInt()

object

  • 数组，对象，函数
